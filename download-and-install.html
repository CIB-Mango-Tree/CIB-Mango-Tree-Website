Download & Install

Programmers can clone our GitHub repository linked here.

Select the file that corresponds to your operating system (Windows, MacOS 14, or MacOS 15).

NOTE: a security popup might appear on first startup warning against running the program. Select "More Information" and select to run the program anyway.

See our Data Preparation page for information on how to prepare your dataset for analysis by the command line interface (CLI).
