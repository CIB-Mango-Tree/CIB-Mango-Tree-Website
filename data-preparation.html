Data Preparation

The CIB Mango Tree does not provide software for scraping social media activity - it yields standardized tests to analyze CIB in data that researchers have already scraped.
Data can be scraped manually from websites using custom-built scrapers, or via APIs that correspond to major social media platforms. The CIB Mango Tree assumes researchers already have datasets of online activity in CSV format that they wish to analyze.
The CIB Mango Tree provides generic tests for CIB. Generic tests can be applied on datasets of online activity from a variety of social media and news sites. CIB Mango Tree tests are optimized for data housed in CSV files, and assumes that CIB can be detected by analyzing a variety of parametersthat datasets of online activity will likely have.
Unique Post Number

A unique number applied to every scraped post/comment

Unique Post Username

The unique username of the account that made the post/comment

Post Date

The date that an individual post/comment was made

Post Time

The time of day that an individual post/comment was made

Post Content

The written content of an individual post/comment

Each CIB Mango Tree test relies on different sets of parameters, depending on the nature of the tests.
Researchers must prepare their data to be run through our tests. Our test programs will detect the content within CSV files by scanning for the titles of each column of data. This means that every column of a data parameter must be headed using the parameter titles above. If columns are not headed using these parameter titles, our test programs will not be able to detect which data must be analyzed.
